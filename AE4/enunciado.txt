Buenos días,

El proyecto consiste en crear un software que sirva para el control de personal,
ebe de utilizar como base SQL en base a Docker.

Nombre de repositorio: ControlHibernate

Se puede dividir el software en tres partes:
Fichar, debe ser un programa que permita asignar una persona con una hora, día y una entrada o salida.
Gestión de los fichajes, Se tiene que mostrar una tabla con la información de los fichajes de cada trabajador,
a tener en cuenta se tiene que guardar y mostrar el clima meteorológico del día.
También tiene que haber un último campo con la suma total de tiempo diario.
Última aplicación que muestre las estadísticas de los fichajes.
Se valorará cualquier aportación extra.

Se tiene que utilizar el Docker ya configurado con el usuario y la contraseña.
